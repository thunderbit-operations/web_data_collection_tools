# Changelog

## 0.1.0 (2024-XX-XX)

### Added
- Core email extraction from text, files, and HTML
- Email deobfuscation engine (bracket, paren, curly, word, HTML entity, URL encoding)
- Recursive web crawling with depth control
- Rate limiting and robots.txt compliance
- Proxy support
- Domain include/exclude filtering
- False positive detection (fake TLDs, placeholder domains)
- Output formats: plain, CSV, JSON, JSONL
- Source tracking (which URL/file each email came from)
- CLI tool with `emx` and `email-harvest` commands
- Python API for library usage
- Stdin pipe support
- Glob pattern support for batch file processing
